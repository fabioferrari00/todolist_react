# ⚛️ Frontend — React ToDo App

Frontend della ToDo List full stack sviluppato con **React + Vite**.

## 🚀 Features
- visualizzazione task
- creazione task
- cambio stato (`todo`, `doing`, `done`)
- eliminazione con modal
- badge dinamici
- FontAwesome icons
- UI responsive

## 🧱 Stack
- React.js
- Vite
- Axios
- CSS custom
- FontAwesome

## ▶️ Avvio
```bash
npm install
npm run dev

🌐 API usate
GET /api/todos
POST /api/todos
PUT /api/todos/:id
